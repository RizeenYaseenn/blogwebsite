Blog Website

A simple and modern blog website developed using HTML, CSS, and JavaScript.
The project focuses on clean design, responsiveness, and basic interactivity without using any external libraries or frameworks.

Features

Responsive blog layout

Clean and modern user interface

Dark mode toggle

Search functionality for blog posts

Read More / Read Less interaction

Technologies Used

HTML

CSS

JavaScript

Usage

Open the index.html file in any modern web browser to view and use the website.
No installation or setup is required.

Purpose

This project was created for frontend practice and learning core web technologies.
It can be used for academic projects, portfolio showcase, and manual testing practice.
